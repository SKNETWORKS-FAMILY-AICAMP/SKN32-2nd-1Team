# 한국미디어패널조사 고객 이탈 예측을 위한 데이터 전처리 결과

## 1. 개요

본 문서는 한국미디어패널조사 데이터를 활용하여 고객 이탈 예측 모형 구축을 위해 수행한 데이터 전처리 과정을 정리한 것이다. 한국미디어패널조사는 동일 표본을 장기간 추적하는 패널조사로, 가구 방문 면접조사 방식으로 수행된다. 본 연구에서는 개인 단위의 연도별 데이터를 결합하여, 다음 조사 시 가입 이동 통신사가 변경되는 경우를 고객 이탈로 정의하였다.

## 2. 데이터 출처 및 조사 범위

데이터는 정보통신정책연구원(KISDI) 미디어통계포털의 한국미디어패널조사 원시자료를 활용하였다. 한국미디어패널조사는 2010년부터 매년 실시되었으며, 전국 17개 시도에서 만 6세 이상 가구원을 대상으로 조사가 이루어진다. 본 전처리에서는 2010년부터 2024년까지의 연도별 자료를 사용하여 패널 형태의 분석용 데이터를 구성하였다.

출처: https://stat.kisdi.re.kr/kor/contents/ContentsList.html

## 3. 고객 이탈 정의

고객 이탈(churn)은 다음 조사에서 가입 이동 통신사가 변경된 경우로 정의하였다. 동일 개인의 현재 연도와 다음 연도의 이동 통신사 정보를 비교하여 값이 다르면 1, 동일하면 0으로 설정하였다. 다만 마지막 조사 연도는 다음 연도 자료가 존재하지 않으므로 이탈 라벨을 산출할 수 없어 결측으로 처리하였다.

## 4. 변수 선정 기준

전 기간에 걸쳐 공통적으로 확보 가능하며 결측이 적은 변수를 우선 선정하였다. 이에 따라 개인 식별 변수, 인구사회학적 변수, 통신비 관련 변수, 결합상품 관련 변수를 중심으로 분석 변수를 구성하였다. 반면 일부 후보 변수는 특정 연도에만 조사되거나 결측 연도가 존재하여 장기 패널 분석에 적합하지 않다고 판단하였다.

## 5. 1차 가공 내용

연도별 원천 데이터를 통합하기 위해 각 연도 자료의 변수명을 일관된 분석용 변수명으로 변경한 뒤, 개인-연도 단위로 결합하였다. 이 과정에서 다음과 같이 변수 체계를 정리하였다.

| 원천 변수      | 변경 변수               | 설명                       |
| ---------- | ------------------- |--------------------------|
| pid        | id                  | 개인 통합 ID                 |
| p__age     | age                 | 나이 카테고리                  |
| p__gender  | gender              | 성별                       |
| p__income  | income              | 소득 카테고리                  |
| p__school  | school              | 학력                       |
| p__hhldsiz | household_size      | 가구원 수                    |
| p__job1    | job                 | 직업 유무                    |
| p__mar     | marriage            | 결혼 카테고리                  |
| p__a02014  | cellular_provider   | 일반휴대폰 가입 이동 통신사(15년~24년) |
| p__a03008  | smartphone_provider | 스마트폰 가입 이동 통신사(15년~24년)  |
| p__a01027  | mobile_provider     | 가입 이동 통신사(10년~14년)       |
| p__c01001  | monthly_total_cost  | 월평균 휴대폰 이용 총 금액          |
| p__c01003  | monthly_installment | 월평균 기기 할부금               |
| p__c02001  | cost_payer          | 휴대폰 요금 부담자               |
| p__c02003  | is_mobile_bundled   | 휴대폰 결합상품 가입 여부           |

또한 조사 연도를 나타내는 year 변수를 추가하여 시계열 분석이 가능하도록 구성하였다.

## 6. 이동 통신사 변수 통합 기준

이동 통신사 변수는 조사 연도에 따라 수집 항목이 상이하므로 단일 변수로 통합하였다. 2010년부터 2014년까지는 mobile_provider를 사용하였고, 2015년부터 2024년까지는 smartphone_provider를 우선 적용하였다. 다만 smartphone_provider가 결측인 경우에는 cellular_provider를 대체 사용하였다.

smartphone_provider와 cellular_provider가 동시에 존재하는 일부 사례에서는 값이 상이한 경우가 있었으나, 해당 사례는 전체 30건으로 매우 적어 전체 분석에 미치는 영향은 제한적이라고 판단하였다. 이에 따라 2015년 이후의 이동 통신사 변수는 smartphone_provider 우선, cellular_provider 보완 방식으로 통합하고 최종적으로 provider 변수로 정리하였다.

## 7. 추가 가공 사항

1. 2010년 데이터에서 월평균 휴대폰 이용 총 금액과 월평균 기기 할부금의 단위가 다른 연도와 상이하여, 다른 연도를 기준으로 단위를 통일하였다.
2. 모름/무응답 범주로 기록된 값은 정보가 없는 응답으로 간주하여 결측과 동일하게 처리하였다.

## 8. 최종 가공 변수

최종 데이터셋은 식별 변수, 시점 변수, 인구사회학적 변수, 통신 이용 및 지출 변수, 그리고 이탈 라벨 변수로 구성하였다.

| 변수명 | 설명                      |
| :-- |:------------------------|
| id | 개인 통합 ID                |
| year | 조사 연도                   |
| age | 나이 카테고리                 |
| gender | 성별                      |
| income | 소득 카테고리                 |
| school | 학력                      |
| household_size | 가구원 수                   |
| job | 직업 유무                   |
| marriage | 결혼 카테고리                 |
| provider | 가입 이동 통신사               |
| monthly_total_cost | 월평균 휴대폰 이용 총 금액(단위: 천)  |
| monthly_installment | 월평균 기기 할부금(단위: 천)       |
| cost_payer | 휴대폰 요금 부담자              |
| is_mobile_bundled | 휴대폰 결합상품 가입 여부          |
| churn | 다음 조사 시 가입 이동 통신사 변경 여부 |

이 중 churn은 지도학습의 라벨 변수이며, 다음 조사에서 가입 이동 통신사가 변경되면 1, 그렇지 않으면 0으로 부여하였다. 마지막 조사 참여 연도는 후속 관측치가 없으므로 결측 처리하였다.

## 9. 카테고리 변수 코딩 기준

카테고리 변수는 원천 데이터의 기준을 유지하였고 다음과 같다.

- age: 1은 만 10세 미만, 2는 만 20세 미만, 3은 만 30세 미만, 4는 만 40세 미만, 5는 만 50세 미만, 6은 만 60세 미만, 7은 만 70세 미만, 8은 만 70세 이상.
- gender: 1은 남, 2는 여.
- income: 1은 소득 없음, 2는 50만원 미만, 3은 100만원 미만, 4는 200만원 미만, 5는 300만원 미만, 6은 400만원 미만, 7는 500만원 미만, 8은 500만원 이상.
- school: 1은 미취학, 2는 초졸 이하, 3은 중졸 이하, 4는 고졸 이하, 5는 대졸 이하, 6은 대학원 재학 이상.
- job: 1은 예, 2는 아니오.
- marriage: 1은 미혼, 2는 결혼, 3은 사별, 4는 이혼.
- provider: 1은 SKT, 2는 KT, 3은 LG U+, 4는 알뜰폰 및 그 외.
- cost_payer: 1은 본인, 2는 회사가 전액 부담, 3는 회사가 일부 지원, 4는 가족이나 타인이 전액 부담, 5는 가족이나 타인이 일부 부담, 6은 기타.
- is_mobile_bundled: 1은 가입, 2는 미가입.

## 10. 모델 입력 기본 변수 구성

모델 입력에는 id를 제외한 모든 설명 변수를 사용하고 churn을 목표 변수로 설정하였다. id는 개인을 식별하기 위한 값으로 예측 신호가 아니며, 특정 개인 고유값을 학습하여 과적합을 유발할 수 있으므로 제외하였다.

## 11. 모델 입력 전처리

1. 마지막 연도인 2024년 데이터는 다음 조사에 대한 이탈 라벨이 없어 학습과 평가에 사용할 수 없으므로 분석 대상에서 제외하였다.
2. 결측치 처리는 변수 유형에 따라 구분하였다. 수치형 변수는 중앙값 대체 후 StandardScaler를 적용하였고, 범주형 변수는 최빈값 대체 후 One-hot 인코딩을 적용하였다.

## 13. 해석상 유의사항

본 데이터는 패널조사 특성상 동일 개인이 여러 해에 걸쳐 반복 관측되므로, 연도 간 비교 시 시점 차이를 함께 고려할 필요가 있다. 또한 마지막 조사 연도는 이탈 라벨이 존재하지 않으므로 학습 및 평가 대상에서 제외하였다. 이동 통신사 통합 과정에서 일부 불일치 사례가 존재하나, 그 규모가 매우 작아 전반적인 분석 결과에 미치는 영향은 제한적이라고 판단하였다.
