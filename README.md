# 📊 고객 이탈 예측 서비스 (Churn Prediction Service)

## 프로젝트 개요

**고객 이탈 예측 서비스**는 머신러닝 기반의 통신사 고객 이탈 위험도 분석 시스템입니다. 한국미디어패널조사 데이터를 활용하여 고객의 인구통계 정보, 서비스 이용 패턴, 결제 이력 등을 분석하고 이탈 가능성을 예측합니다. 이를 통해 통신사는 고위험 고객에게 사전에 맞춤형 혜택을 제공하여 이탈을 방지할 수 있습니다.

---

## 🎯 주요 기능

### 1. 고객 정보 입력 및 분석
- **기본 정보:** 성별, 나이, 학력, 거주 지역
- **통신 서비스 정보:** 통신사, 월 이용 요금, 가입 기간, 요금제 유형
- **이용 패턴:** 앱 이용 여부, 최근 활동도, 연체 여부

### 2. 머신러닝 기반 이탈 예측
- **알고리즘:** Random Forest, XGBoost, LightGBM
- **예측 결과:** 이탈 확률(%), 위험도 판정(저/중/고)
- **신뢰도:** 모델 성능 평가 지표 포함

### 3. 시각화 및 인사이트
- **이탈 확률 게이지 차트:** 직관적인 위험도 표시
- **Feature Importance:** 이탈에 영향을 미치는 주요 요인 시각화
- **고객 프로필:** 입력된 정보 요약 및 비교

### 4. 맞춤형 권장 조치
- **고위험군:** 긴급 고객 관리, 특별 혜택 제공
- **중위험군:** 정기적 모니터링, 예방적 조치
- **저위험군:** 현상 유지, 장기 고객화 전략

## 🚀 설치 및 실행

### 1. 필수 요구사항
- Python 3.11 이상
- pip (Python 패키지 관리자)

### 2. 설치 단계

#### Step 1: 저장소 클론 또는 파일 다운로드
```bash
git clone https://github.com/your-repo/churn-prediction-service.git
cd churn-prediction-service
```

#### Step 2: 가상 환경 생성 (선택사항이지만 권장)
```bash

```

```bash

```

### 3. 실행

#### 웹 애플리케이션 실행

## 🔬 모델 성능

### 학습 데이터 기준 (2018년 데이터)

| 모델 | 정확도 (Accuracy) | F1-Score | 재현율 (Recall) |
| :--- | :--- | :--- | :--- |
| **Random Forest** | 83.6% | 0.158 | 0.11 |
| **XGBoost** | 85.7% | 0.157 | 0.10 |
| **LightGBM** | 86.4% | 0.079 | 0.04 |

**주요 성능 지표 설명:**
- **정확도:** 전체 예측 중 맞춘 비율
- **F1-Score:** 정밀도와 재현율의 조화평균 (이탈자 예측 정확도)
- **재현율:** 실제 이탈자 중 올바르게 예측한 비율

### 주요 이탈 영향 요인 (Feature Importance)
1. **월 이용 요금** (28%) - 요금이 급감하는 고객이 이탈 위험 높음
2. **최근 활동도** (22%) - 최근 서비스 이용이 적을수록 이탈 가능성 높음
3. **가입 기간** (18%) - 초기 가입자의 이탈률이 높음
4. **요금제 유형** (15%) - 저가형 요금제 사용자의 이탈률 높음
5. **연체 여부** (12%) - 연체 이력이 있는 고객의 이탈 위험 높음

## 🔧 결측치(Missing Value) 처리 전략

데이터셋에 포함된 결측치는 다음과 같이 처리됩니다:

| 변수 유형 | 처리 방법 | 사유 |
| :--- | :--- | :--- |
| **인구통계 (나이, 성별)** | 제거 또는 최빈값 대체 | 신뢰도 유지 필수 |
| **이용 금액** | 0으로 대체 | 미이용 = 0원 의미 |
| **통신사/요금제** | 'Unknown' 카테고리 생성 | 미가입 상태를 정보로 활용 |
| **이용 패턴** | 중앙값(Median) 대체 | 데이터 분포 왜곡 최소화 |
